## 专业学习
由于以后研究生学习可能会涉及到Linux开发，所以我选择了《Linux就这么学》这本书。由于linux的开源特性，我甚至可以在官网上找到这本书的开源电子版（网页版）。前两章主要介绍了Linux的起源、开源文化以及最基本的系统安装与命令操作。第一章我学习了Linux为何在服务器与开发领域占据主导地位，尤其是其稳定性和自由修改的特性。第二章则从实际操作出发，学习了ls、cd、pwd等基础命令的使用，同时使用wsl亲手在虚拟机上完成了简单的练习。虽然这些命令看似简单也很像Windows命令行操作，但它们构成了后续复杂操作的地基。通过这两章的学习，我对Linux的基本交互方式有了初步的认识，也为后续学习Shell脚本和系统编程打下了必要基础。

## 实验安排及结果产出展示
首先选择要进行对比实验的模型，根据获取源码的便携程度、模型的类别以及根据一篇文章中的实验数据，总共选取了三个模型：AGPCNet、RDIAN、UiUNet。
由于选取的模型都有便于获取的源码和已经依据其论文训练好的pt或pth文件，原本训练使用的数据集和参数等不同于本实验提出的模型及训练集，所以，需要对选取的模型修改训练代码，使用相同的训练集和分割比例。
所有的训练都在本地进行，因为本地平台的Nvidia RTX3070ti(8G)已经足够训练非通用大模型的训练，只是训练时间稍微变长一点。实际上在进行训练的时候，并未花费过多时间。
难点在于保持对比模型的评测指标和本实验提出的模型方法的评测指标完全相同。不同模型的输入输出不尽相同，而选取的评测指标要提前导入模型结构并稍加修改部分参数，如输入的图片和淹没图片的格式等。
